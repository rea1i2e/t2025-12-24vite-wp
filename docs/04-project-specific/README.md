# 案件固有情報

このセクションは、**案件ごとに記入する情報**を管理するためのテンプレートです。

## このセクションのドキュメント

- **[overview.md](overview.md)**: 案件仕様（テンプレート）
  - プロジェクト情報、技術スタック、ディレクトリ構成
  - コンポーネント一覧、投稿タイプ、ページ構成
  - テンプレートファイル、機能一覧など

## 使い方

### 新規案件作成時

1. テンプレートリポジトリから新規リポジトリを作成
2. `overview.md` を開き、案件固有の情報を記入
3. プロジェクト情報、技術スタック、コンポーネント一覧などを記入

### 記入項目

- **プロジェクト情報**: プロジェクト名、クライアント名、期間など
- **技術スタック**: WordPress、PHP、Node.js、Viteのバージョンなど
- **ディレクトリ構成**: 案件固有のディレクトリがあれば記載
- **コンポーネント一覧**: PHPコンポーネント、JavaScriptモジュール、Sassコンポーネント
- **投稿タイプ**: デフォルト投稿タイプとカスタム投稿タイプ
- **ページ構成**: ページ名、スラッグ、ページ種別など
- **テンプレートファイル**: テンプレートファイルの一覧
- **機能一覧**: WordPress機能、カスタム機能
- **画像最適化設定**: WebP生成、品質設定など
- **デプロイ設定**: デプロイ先、FTPサーバー、Discord通知など
